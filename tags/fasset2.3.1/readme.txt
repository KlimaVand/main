News in fasset 2.3.1
Improved performance in the soil part of the model (around 20%, depends….)
Fasset will now make warnings for the information in the input files it has not used (eg only for those actually open).
Fasset will make a list of files it tried to read in fileList.txt. It will be saved next to fasset self.
MinGW is now supported without any conversion of input files 
Fasset can now be compiled and run under 64 bit with around 10 % speedup.
