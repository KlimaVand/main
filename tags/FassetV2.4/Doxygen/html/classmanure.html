function to calculate gaseous emissions of N and mineralisation of organic N, using a factor approach

Parameters:
NH3EFemission factor for ammonia
N2EFemission factor for dinitrogen
N2OEFemission factor for nitrous oxide
MinFproportion of organic N that is mineralised
*NH3volpointer used to return the amount of ammonia volatilised
*N2volpointer used to return the amount of dinitrogen volatilised
*N2Ovolpointer used to return the amount of nitrous oxide volatilised
*MinNpointer used to return the amount of organic nitrogen mineralised

calculate how much NH3, N2,N2O and N there is evaporate. Those value is return in NH3vol, N2vol, N2Ovol, MinN NH3EF, N2EF, N2OEF, MinF is the fraction of NH4 that is transformed in the process. It is then removed from the manure
